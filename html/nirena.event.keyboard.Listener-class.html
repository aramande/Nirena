onKeyDown(self, code, keys=None)

source code 

Stores 'code' to be called once when the user pushes down one of 'keys' then calls the code with a keyboard.Event object as argument.

Parameters:
  • code - Expects a function that takes one argument for the keyboard.Event object
  • keys - A list of pygame keycodes; if an entry is a list or tuple, first value should be a modifier and second a key; if None, code will always be called when a key is pushed

whileKeyPressed(self, code, keys=None)

source code 

Stores 'code' to be called every frame from when the user pushes down one of 'keys' until the user releases the same key then calls the code with a keyboard.Event object as argument.

Parameters:
  • code - Expects a function that takes one argument for the keyboard.Event object
  • keys - A list of pygame keycodes; if an entry is a list or tuple, first value should be a modifier and second a key; if None, code will always be called while a key is being pushed

onKeyUp(self, code, keys=None)

source code 

Stores 'code' to be called when the user releases a previously pressed key and then calls the code with a mouse.Event object as argument.

Parameters:
  • code - Expects a function that takes one argument for the keyboard.Event object
  • keys - A list of pygame keycodes; if an entry is a list or tuple, first value should be a modifier and second a key; if None, code will always be called when a key is released
