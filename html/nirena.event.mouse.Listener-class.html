onButtonDown(self, code, button=None)

source code 

Stores 'code' to be called when the user clicks a button on the mouse and then calls the code with a mouse.Event object as argument.

Parameters:
  • code - Expects a function that takes one argument for the mouse.Event object
  • button - The identification of the button on the mouse, if None, code is run every time the user clicks

whileButtonPressed(self, code, button=None)

source code 

Stores 'code' to be called every frame when the user holds a button on the mouse and then calls the code with a mouse.Event object as argument.

Parameters:
  • code - Expects a function that takes one argument for the mouse.Event object
  • button - The identification of the button on the mouse, if None, code is run every frame the user clicks any mousebutton

onButtonUp(self, code, button=None)
