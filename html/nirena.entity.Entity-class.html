Creates an entity with their own sprite and position on the screen.

Parameters:
  • layer - An integer telling the world what layer the entity will be visible on
  • scale - A decimal coeficient to multiply size by
  • trans - Opaque value of the image from 1-255 where 255 is completely opaque
  • tint - Tints the sprite into another color. Not sure how this works, implement on own risk
  • if_wrap - Not sure how this works or what it does, implement on own risk
  • region_size - Not sure how this works or what it does, implement on own risk

flip(self, x)

source code 

Flips the sprite in one direction or the other. Useful replacement for a turning animation.

Parameters:
  • x - Expects a boolean value, True to flip on x-axel, False to flip on y-axel
Raises:
  • ValueError - If x is anything but a boolean value
